# Project Overview 🎉
Website about photography

# Technologies 🔧

- Typescript
- React
- Scss

# Installation 💾

1. Clone the repository `git clone https://github.com/MattGri/photosnap-website.git`
2. Install dependencies `npm install`
3. At the terminal run the command `npm start`
